Live Web Region

Websites (URL oder lokale HTML-Datei) live in PowerPoint anzeigen – auch im Präsentationsmodus.

In 3 Schritten

  1. Link setzen: Reiter Live Web → Link setzen. Ist eine Form markiert, wird sie verwendet. Ist nichts markiert, wird automatisch ein Rechteck eingefügt. Anschließend eine URL (https://…) eingeben oder eine lokale HTML-Datei wählen.
  2. Position/Größe der markierten Form nach Wunsch anpassen – der Web-Bereich folgt der Form.
  3. Präsentation starten (F5): Im Bereich läuft die Seite live und interaktiv.

Im Präsentationsmodus

  • Interaktion: Mausklicks, Tippen, Buttons funktionieren direkt auf der Seite.
  • Weiterblättern / Beenden: → ← Bild ↑/↓ und Esc steuern die Präsentation – auch dann, wenn die Seite gerade den Fokus hat.

Tipp: Mehrere Bereiche pro Folie sind möglich. Jeder Bereich kann eine eigene HTML-Datei zeigen.

Verwalten

  • Datei ändern: markierte Form auf eine andere URL/HTML-Datei umstellen.
  • Link entfernen: Web-Zuordnung der markierten Form löschen.
  • Neu laden: Web-Inhalte der aktuellen Folie neu laden.

Voraussetzungen

Windows · PowerPoint Desktop (64-Bit) · Microsoft Edge WebView2 Runtime (auf aktuellen Windows-Systemen meist vorinstalliert).
